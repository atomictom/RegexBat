RegexBat
========

Practice Regular Expressions!
-----------------------------

This is a website intended to allow people to learn how to write regular expressions and practice doing so. It's one thing to "know" how to do something. It's another to be able to do it (and be comfortable doing it!).

Status:
-------

I have a rough draft of how the front-end should look, but haven't done anything for the backend yet. I plan to pick this up again when I'm more comfortable with how to do backends (or feel like I have time to learn with this).
